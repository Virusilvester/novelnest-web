Getting started

Install in minutes

1

Download the APK

Tap the Download button above to grab the latest release APK directly to your Android device.

2

Allow unknown sources

Go to Settings → Apps → Special app access → Install unknown apps and enable it for your browser or file manager.

3

Install and add sources

Install the app, then browse the source catalog to add your favorite light novel and web novel platforms.

4

Start reading

Search for novels, add them to your library, and download chapters for offline reading. Happy reading!
